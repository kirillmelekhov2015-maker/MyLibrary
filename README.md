This is my first project. Please don't judge it too harshly. At the moment, the project is only available in Russian, but I will add English in the future.

This application is a small library combined with notes. All files are in .md format and can be exported and imported. It is also possible to disable notes or other tabs (Books, Anime, Manga, and TV Series).
Statistics are available for existing works; notes are not included in the statistics.

Приложение представляет собой небольшую библиотеку, с возможностью добавлять заметки. Все файлы в формате .md и могут быть экспортированы и импортированы. Также можно отключить заметки или другие вкладки (книги, аниме, манга и сериалы). Доступна статистика по существующим произведениям, заметки в статистику не включаются.
